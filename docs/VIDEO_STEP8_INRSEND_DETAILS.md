# Étape vidéo 8 — Détails iNrSend des variantes vidéo

Cette étape finalise la lecture des vidéos adaptées dans iNrSend.

## Objectif

Quand une publication vidéo est ouverte dans iNrSend > Publications > Détails, l'app affiche désormais la vidéo réellement utilisée par le canal actif, et non seulement la vidéo source.

## Données affichées

Pour chaque canal vidéo :

- canal actif ;
- format final utilisé : 9:16, 1:1, 16:9 ou Original ;
- mode d'adaptation : fond flouté sécurisé ou recadrage plein écran ;
- nom du fichier vidéo final ;
- poids et durée quand disponibles ;
- chemin de stockage quand disponible ;
- vidéo source conservée quand elle est différente de la variante finale.

## Comportement

- Le détail vidéo est visible même hors mode modification.
- En mode modification, la vidéo finale est conservée et seuls les textes, CTA et hashtags restent modifiables en V1.
- Les anciennes publications restent compatibles : si aucune variante n'existe, iNrSend affiche la vidéo disponible.
